Project Overview

AKTIV is a modern online workshop platform designed to provide professional skill development and training opportunities. The platform offers a comprehensive booking system, workshop management, payment processing, and user-friendly interfaces for both workshop organizers and participants. Built with modern web technologies, it provides a seamless experience across all devices.

Project Setup

Prerequisites

  • VS Code (Visual Studio Code)
  • Node.js (for installing dependencies with npm)
  • Live Server Extension for VS Code

Installation Steps

  1. Download ZIP file:

    Download the AKTIV Workshop project ZIP file from the repository or available source.

  2. Extract ZIP file:

    Extract the downloaded ZIP file to your desired folder on your computer.

  3. Open extracted folder:

    Navigate to the extracted folder to view the project structure.

  4. Open project folder in VS Code:

    Right-click on the project folder, then select "Open with Code" or open VS Code and drag the folder into VS Code.

  5. Install dependencies:

    Follow these steps to install the required dependencies:

    • Open VS Code terminal (press Ctrl+` or go to Terminal ‚Üí New Terminal)
    • Run the command: npm install
    • Wait for the installation to complete (this will install Tailwind CSS and Prettier)
  6. Install Live Server Extension:

    Follow these steps to install the Live Server Extension:

    • Open VS Code
    • Click on the "Extensions" icon in the left sidebar (or press Ctrl+Shift+X)
    • In the search box, type "Live Server"
    • Find the "Live Server" extension by Ritwick Dey
    • Click the "Install" button on that extension
    • Wait for the installation process to complete
    • The extension will be automatically activated after installation is complete
  7. Start Tailwind CSS compiler:

    Run npm run dev in the terminal to start the Tailwind CSS compiler in watch mode.

  8. Open landing page:

    Navigate to any HTML file in src/pages/ and right-click to select "Open with Live Server". The browser will automatically open and display the AKTIV workshop platform.

Note: Live Server will automatically reload the page when you make changes to HTML, CSS, or JavaScript files. The server will run at http://127.0.0.1:5500 or an available port.
Tips: Make sure you have installed VS Code first before following the steps above. If you encounter issues with Live Server, try restarting VS Code or check if the port being used is not already in use by another application.

How to Edit/Modify the Project

Making Changes to the Project

Once you have the project running with Live Server, you can start editing and modifying the code. Here's how to make changes effectively:

File Structure for Editing

  1. HTML Files:

    All HTML pages are located in the src/pages/ directory. You can edit any HTML file directly in VS Code. Changes will be automatically reflected in the browser thanks to Live Server.

    • src/pages/index.html - Login page
    • src/pages/head-village/ - Village head pages
    • src/pages/head-family/ - Head of household pages
  2. CSS Styling:

    The project uses Tailwind CSS. You can modify styles in two ways:

    • Tailwind Classes: Add Tailwind utility classes directly in HTML
    • Custom CSS: Add custom styles in src/css/input.css
    • Compiled CSS: The final CSS is generated in src/css/output.css
    • Important: You must run npm run dev to compile Tailwind CSS changes
  3. JavaScript Functionality:

    JavaScript files are located in src/js/ directory. Each file handles specific functionality:

    • sidebar.js - Navigation functionality
    • modal.js - Popup dialogs
    • tabs.js - Tab navigation
    • upload-image.js - Image upload features
    • And more...
  4. Images and Icons:

    Static assets are stored in the public/ directory:

    • public/icons/ - SVG icons for the interface
    • public/images/ - PNG images and graphics

Development Workflow

  1. Start Tailwind CSS Compiler:

    IMPORTANT: Before making any changes, you must start the Tailwind CSS compiler by running npm run dev in the VS Code terminal. This will watch for changes and automatically compile your CSS.

  2. Start Live Server:

    Make sure Live Server is also running so you can see changes in real-time in the browser.

  3. Edit Files:

    Open any file you want to modify in VS Code and make your changes. You can edit HTML, CSS, or JavaScript files.

  4. Save Changes:

    Save the file (Ctrl+S). Both the Tailwind compiler and Live Server will automatically process and reload your changes.

  5. Test Changes:

    Check your changes in the browser to ensure everything works as expected.

Critical: You MUST run npm run dev in the terminal before making any modifications to the project. This command starts the Tailwind CSS compiler which watches for changes and automatically compiles your CSS. Without this, your Tailwind CSS changes will not be applied to the project.
